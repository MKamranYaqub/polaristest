This document outlines the proposed implementation of a Multi-Property Portfolio section within the Bridging Calculator. The feature will enable brokers to input multiple properties in a portfolio deal and automatically calculate blended rates based on property values, LTV tiers, charge types, and property classifications.

How It Works

Blended Rate Formula

Blended Rate = Œ£ (Property Value √∑ Total Portfolio Value) √ó Rate for that Property

Each property's contribution to the blended rate is proportional to its value in the portfolio. Properties with higher values have more influence on the final rate.
